# セレクタ

CSSには、 **セレクタ** というものがあります。

イメージするとすれば、英語の5W1Hで言う「where」に該当します。

スタイルを付与する場所（where）を意味するのがセレクタです。

セレクタには名前がつきます。

これを **セレクタ名** と言います。

セレクタ名は、自由につけられません。

セレクタ名は、すでに存在しているHTMLファイル内の以下のものを、基本的に使用します。

* class属性（class名）
* id属性（id名）
* 要素名（タグ名）

セレクタ名を書く表記方法で、これら3つの区別をつけなければわかりにくいですね。

それぞれのセレクタ名の書き方は以下です。

```css
/* class属性 */
.main-body

/* id属性 */
#top-image

/* 要素名 */
h1
```

HTML要素 or 属性名 | セレクタ名に使用する記号
------------ | -------------
class属性 | .（ドット）
id属性 | #（シャープ）
要素名（タグ名） | 無し
